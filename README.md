# HHYun
简单的个人网盘（仿百度网盘）,学习SSM框架的项目，在[CSDN](https://download.csdn.net/download/weixin_39793646/10729199)上找的项目，向原作学习（这里仅有部分功能，原作功能很强大，支持原作）。在学习过程中做了如下修改：
* 原作为struts2，这里用的springMVC
* 原作在主页上做了自己的样式，这里用的主要是模仿百度网盘
* 图片预览用的viewer.js，视频预览用的video.js
* 非阻塞消息提示框toastr.js
* 上传组件webuploader，组合了spark-md5.js实现秒传
* 用easyui做拖拽式网格布局，样式没调好

## 功能
* 文件的上传（秒传）、下载、在线预览和分享（公开或加密、可设有效期）功能
* 视频可倍速观看
* 网盘内容可按文件的修改日期、文件名、大小排序显示
* 还有回收站功能
